GitHub Pages Feed Hub

IMDb watchlists and lists, rebuilt as Radarr-friendly RSS.

This site publishes static RSS feeds that Radarr can import with its RSS List source. Each feed is refreshed by GitHub Actions and filtered down to movie-like IMDb title types.

How To Use It

Open one of the XML links below and paste that URL into Radarr as an RSS List source.

  1. Choose a feed card below.
  2. Copy the XML URL.
  3. In Radarr, add an import list of type RSS List.
  4. Paste the feed URL and save.
